# Canvas Planner for Mac

A local tool to sync your Canvas assignments into a clean Excel schedule.

### Setup
1. Download and unzip the folder.
2. Right-click `start_planner.command` and select **Open**.
3. Your browser will open with the interface. Paste your API key to begin!

### Privacy
This app runs entirely on your local machine. Your API keys are never stored or sent anywhere except to the Canvas API.
